В представленном проекте мной были выполнены следующие задачи:
	сбор, формализация и декомпозиция требований к системе
	наполнение бэклога задач при разработке по Agile Kanban
	участие в построении архитектуры системы
	сквозное тестирование системы
	настройка, анализ и оптимизация БД Elasticsearch
	конфигурирование системы ELK (в том числе Ruby фильтры Logstash)
	определение и формализация log записей системы мониторинга
	реализация компонента мониторнга, язык C#
	реализация высоконагруженного компонента взаимодействия с БД RIPE, язык Java
